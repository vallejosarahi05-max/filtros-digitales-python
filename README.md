# Implementación y evaluación de filtros digitales

Este repositorio contiene el desarrollo de la Actividad Formativa 3, cuyo objetivo es diseñar, implementar y evaluar filtros digitales pasa bajas, pasa altas y pasa bandas utilizando el lenguaje de programación Python.

El proyecto incluye la generación de una señal compuesta con ruido blanco, el diseño de filtros digitales tipo Butterworth y la aplicación de dichos filtros para analizar su comportamiento en el dominio del tiempo y en el dominio de la frecuencia. Las gráficas obtenidas permiten comparar la señal original con las señales filtradas y verificar la efectividad de cada filtro.

## Contenido del repositorio
- filtros_digitales.py: Código fuente en Python para la implementación de los filtros digitales.
- README.md: Descripción del proyecto y explicación general del desarrollo.

## Herramientas utilizadas
- Python
- NumPy
- SciPy
- Matplotlib

## Resultados
Los resultados demuestran que los filtros digitales son herramientas eficaces para el análisis de señales, ya que permiten eliminar ruido y separar componentes de frecuencia de manera clara y eficiente.
